# SECprofiler

## Setup
    $ git clone https://github.com/hafenr/SECprofiler
    $ cd SECprofiler
    $ R
    >> require(devtools)
    >> load_all()
    
## Workflow

After updating the roxygen docs issue the command

    >> document()
